# Robot Testcases

## 项目简介
这是一个面向扫地机器人测试场景的测试用例与测试设计示例仓库，用于展示智能硬件测试、场景测试和测试分析能力。

## 覆盖模块
- 导航建图
- 避障能力
- 自动回充
- App配网与控制
- 异常场景分析

## 文档说明
- navigation_testcases.md：导航与建图测试用例
- obstacle_avoidance_testcases.md：避障测试用例
- recharge_testcases.md：自动回充测试用例
- bug_summary.md：缺陷示例与问题分析

## 说明
该仓库用于展示机器人测试思路、测试设计方法和场景覆盖能力。
